# 📄 Hoja de Ejercicios 04 - CSS Avanzado, BEM y Flexbox

Esta carpeta contiene la resolución de la cuarta hoja de ejercicios del bootcamp, centrada en consolidar técnicas avanzadas de CSS, alineación y metodologías de nomenclatura.

## 🚀 Fases y Ejercicios Completados

### Fase 1: Selectores y Estados
* **Ejercicio 1 - Combinadores y atributos:** Uso de selectores avanzados (hijo directo `>`, hermano adyacente `+`) y selectores de atributo (`[type="email"]`, `[required]`) para dar estilo sin añadir clases extra.
* **Ejercicio 2 - Pseudo-clases:** Implementación de interactividad y estados de estructura usando `:hover`, `:focus-visible` (accesibilidad por teclado), `:checked`, `:disabled`, `:first-child` y `:nth-child()`.

### Fase 2: Modelo de Caja y Desbordamiento
* **Ejercicio 3 - Pseudo-elementos:** Uso de `::before` y `::after` para añadir decoraciones visuales sin ensuciar el HTML, junto con personalización de `::placeholder` y `::selection`.
* **Ejercicio 4 - Box Model:** Comprensión práctica de las dimensiones en CSS, controlando las diferencias entre `margin`, `padding` y aplicando `box-sizing: border-box` globalmente.
* **Ejercicio 5 - Overflow y Display:** Control de contenido desbordado con `overflow: auto/hidden` y manipulación del flujo del documento alternando entre `block`, `inline` e `inline-block`, además de usar `display: none`.

### Fase 3: Flexbox y BEM
* **Ejercicio 6 - Flexbox Básico:** Primera toma de contacto con ejes y alineaciones, utilizando `justify-content`, `align-items`, `gap` y `flex-wrap` para distribuir elementos de navegación y tarjetas.
* **Ejercicio 7 - Metodología BEM y Nesting:** Adopción de la convención de nomenclatura Block-Element-Modifier (BEM) para crear clases escalables, combinada con el moderno CSS Nesting nativo (`&`) para mejorar la legibilidad de la hoja de estilos.

### Fase 4: Proyecto Final
* **Ejercicio 8 - Panel de Cursos:** Integración de todos los conceptos anteriores en una interfaz de usuario completa y coherente. El proyecto incluye un layout completo en Flexbox, un formulario de contacto, una sección de FAQ funcional creada con `<details>` y código CSS estructurado bajo la metodología BEM.

## 🛠️ Stack Tecnológico
* HTML5 (Etiquetas semánticas, accesibilidad básica).
* CSS3 (Flexbox, CSS Nesting nativo, Custom Scrollbars/Selection).
* Git (Flujo de ramas por fases y *Conventional Commits*).
