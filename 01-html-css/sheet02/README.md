# Hoja de Ejercicios 02 - HTML II

Este directorio contiene los ejercicios correspondientes a la segunda sesión de HTML del bootcamp. El enfoque principal de esta fase es aprender a estructurar datos complejos y construir interfaces de usuario interactivas de forma semántica y accesible utilizando únicamente HTML5 nativo.

## 🎯 Competencias trabajadas

A lo largo de estos 11 ejercicios se ponen en práctica los siguientes conceptos:
- Construcción de **formularios** con estructura clara.
- Elección correcta de tipos de controles (`input`, `select`, `textarea`, etc.) según el dato esperado.
- Accesibilidad práctica y relación correcta entre `label`, `for`, `id` y `name`.
- Aplicación de **validación nativa** (`required`, `minlength`, etc.).
- Agrupación semántica de datos mediante `fieldset` y `legend`.
- Construcción de **tablas de datos** accesibles diferenciando cabeceras de fila y columna (`caption`, `th`, `scope`).
- Uso de elementos nativos de interfaz como `details`, `summary`, `progress`, `meter` y `dialog`.

## 📂 Estructura del proyecto

Los ejercicios están divididos en un flujo de trabajo progresivo:

- **exercise-01:** Formulario básico de inscripción.
- **exercise-02:** Formulario de creación de cuenta (validación nativa).
- **exercise-03:** Formulario con grupos semánticos (elección única y múltiple).
- **exercise-04:** Formulario de búsqueda y filtros.
- **exercise-05:** Tabla básica de datos.
- **exercise-06:** Tabla avanzada con encabezados de fila.
- **exercise-07:** Preguntas frecuentes (`details` y `summary`).
- **exercise-08:** Panel de progreso y estado (`progress` y `meter`).
- **exercise-09:** Diálogo y contenido incrustado (`dialog` e `iframe`).
- **exercise-10:** Corrección y auditoría de un formulario defectuoso.
- **exercise-11:** Mini proyecto integrador final.

## 🛠️ Flujo de trabajo

El desarrollo de esta hoja se ha realizado utilizando control de versiones con **Git**, separando el trabajo en ramas funcionales (`feature/*`) y aplicando el estándar **Conventional Commits** para mantener un historial limpio y descriptivo.
