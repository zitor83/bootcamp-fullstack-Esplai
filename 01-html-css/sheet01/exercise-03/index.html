Metodología y Nivel

¿Cuál es la duración del curso?

El curso tiene una duración de 4 semanas, con clases de 2 horas cada semana.

¿Se necesita experiencia previa?

No, el curso está diseñado para principiantes, por lo que no se requiere experiencia previa en cocina japonesa.

Materiales y Precios

¿Qué materiales se proporcionan?

Se proporcionan todos los ingredientes necesarios para las recetas, así como un manual de cocina con instrucciones detalladas.

¿Cuál es el precio del curso?

El precio del curso es de 300 euros, que incluye todos los materiales y el acceso a las clases.
