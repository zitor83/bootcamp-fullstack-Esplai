Por qué conviene escribir HTML semántico

El HTML semántico es una forma de escribir código HTML que utiliza etiquetas que tienen un significado claro y específico. Esto también hace que mejore la accesibilidad para personas con discapacidades y ayuda a a los motores de búsqueda a entender mejor el contenido de una página web.

Mejora la accesibilidad y el SEO

Se suele decir en desarrollo web: Las buenas etiquetas son como buenos cimientos en un edificio. Hazlo bien desde el principio.

"HTML es la piedra angular de la web"

— José Ortiz

Ejemplo de código

Es muy común ver páginas que usan mal las etiquetas. Por ejemplo, en lugar de usar la etiqueta <div> para crear un menú, deberíamos usar la etiqueta <nav>.

Fíjate en este ejemplo de cómo NO estructurar un título:


<div class="titulo-principal">
    Mi Página Web
</div>
