# Hoja de Ejercicios 1: Fundamentos de HTML 🧱

Este directorio contiene las soluciones a la primera hoja de ejercicios del bootcamp, centrada en asimilar la estructura básica y las etiquetas fundamentales de HTML5.

## 📝 Conceptos Trabajados

A lo largo de estos ejercicios iniciales, hemos puesto en práctica la creación de documentos web semánticos desde cero:

* **Estructura base:** Comprensión de las etiquetas `<html>`, `<head>`, `<body>` y la configuración del `viewport` y codificación de caracteres (`UTF-8`).
* **Jerarquía de texto:** Uso correcto de los encabezados (`<h1>` a `<h6>`) para estructurar la información y párrafos (`<p>`).
* **Listas:** Creación de listas ordenadas (`<ol>`) y desordenadas (`<ul>`) con sus respectivos elementos (`<li>`).
* **Enlaces e Imágenes:** Navegación entre páginas usando la etiqueta `<a>` con sus atributos `href` y `target`, e inserción de contenido multimedia con la etiqueta `<img>` y su atributo `alt` para accesibilidad.
* **Formateo básico:** Aplicación de énfasis y negritas de forma semántica con `<em>` y `<strong>`.
